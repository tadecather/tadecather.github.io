设置一个酷炫的主题，让你自己的博客与众不同！

设置 hexo 的主题设置十分方便。
前往hexo的主题预览页面 Hexo/themes，然后便可以看到不同
的主题的预览。点击预览图片，可以进入到一个使用了该主题的网页。

点击下面的主题名字进入到该主题的 github 页面。

需要 clone 该项目到你自己的博客文件夹下的 themes 目录下。
在你的博客文件夹目录下打开命令框，然后执行 git clone 操作。

git clone 你比较喜欢的主题的网址 themes/主题名字

例如这个:
git clone https://github.com/levblanc/hexo-theme-aero-dual themes/aero-dual

这个是我的博客所用的主题。 area-dual
一般来说，是需要更新一下的主题的。进入到刚刚 clone 的目录下，执行以下命令:
git pull

然后更改 [yourblogdirector/config.yml] 文件的 themes 标签为你刚刚保存项目的目录即可。
另外根据不同的主题可以设置主题的具体表现形式。

关于主题的具体设置，可以查看该主题的 github 的 README.md 文件进行设置。
